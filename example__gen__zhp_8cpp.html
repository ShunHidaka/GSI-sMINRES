Detailed Description

Demo: Generalized shifted MINRES with ZHP storage and LAPACK packed-Cholesky inner solves.

Solves \((A + \sigma^{(m)} B)x^{(m)} = b\) with A and B read into BLAS packed-Hermitian (ZHP) storage. The inner solves with \(B^{-1}\) are performed by zpptrf/zpptrs (packed-Cholesky), while zhpmv applies A and B.

Input

  • A.mtx (Hermitian / symmetric, ZHP after loading)
  • B.mtx (Hermitian positive definite, ZHP after loading)
  • The program enforces uploA == uploB to keep storage consistent.

Output (per shift m)

  • index, Re(σ), Im(σ), iteration count, algorithmic residual estimate, and the true 2-norm residual \(\|(A+\sigma B)x-b\|_2\).
Usage
a_gen_zhp.out A.mtx B.mtx
Note
Requires BLAS/LAPACK; uses zpptrf/zpptrs for \(B^{-1}\) and zhpmv for matrix-vector products.
