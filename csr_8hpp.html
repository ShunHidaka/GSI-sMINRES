Detailed Description

Minimal CSR (Compressed Sparse Row) struct used by GSI-sMINRES sample programs.

Author
Shuntaro Hidaka

Plain data holder for a square sparse matrix in CSR format. Indices are 0-based. The structure does not enforce invariants at runtime; callers are responsible for providing consistent arrays.

Expected invariants (for correctness and performance):

  • row_ptr.size() == n + 1
  • col_idx.size() == values.size() (== nnz)
  • row_ptr is non-decreasing and row_ptr[0] == 0, row_ptr[n] == nnz
  • 0 <= col_idx[k] < n for all 0 <= k < nnz
  • (Recommended) column indices in each row are strictly increasing to enable fast searches/merges.

Notes:

  • This struct does not imply any symmetry/hermiticity.
  • No memory ownership beyond std::vector; no methods are provided on purpose.
