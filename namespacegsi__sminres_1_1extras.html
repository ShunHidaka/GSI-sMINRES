Solve \( Ax=b \) using the (unpreconditioned) Conjugate Gradient (CG) method.

Iteration stops when the relative residual \( \|r_k\|_2 / \|b\|_2 \le \mathrm{rtol} \) or when max_iter iterations are reached. On exit, x always contains the last iterate (useful even if not converged).

Parameters
[in]ACoefficient matrix (real-symmetric or Hermitian positive definite), stored in CSR format as defined in gsi_sminres/extras/sparse/csr.hpp.
[in,out]xSolution vector; on input an initial guess, on output the approximate solution.
[in]bRight-hand side vector.
[in]max_iterMaximum number of iterations.
[in]rtolRelative residual tolerance.
[out]itersNumber of iterations performed (set even if the method does not converge).
[out]relresFinal relative residual \( \|r\|_2 / \|b\|_2 \) at exit.
Returns
true if the tolerance is satisfied within max_iter; false otherwise.
Precondition
A is stored in CSR format as defined in gsi_sminres/extras/sparse/csr.hpp.
A is square and SPD/HPD.

Solve \( (A + \omega B)x=b \) using the (unpreconditioned) MINRES method.

Iteration stops when the relative residual \( \|r_k\|_2 / \|b\|_2 \le \mathrm{rtol} \) or when max_iter iterations are reached. On exit, x always contains the last iterate (useful even if not converged).

Parameters
[in]ACoefficient matrix \( A \) (real-symmetric or Hermitian), stored in CSR format as defined in gsi_sminres/extras/sparse/csr.hpp.
[in]omegaShift parameter \( \omega \) (accepted as complex for BLAS convenience, but MINRES requires \( \omega \in \mathbb{R} \) so that \( A + \omega B \) is real-symmetric or Hermitian).
[in]BCoefficient matrix \( B \) (real-symmetric or Hermitian), stored in CSR format as defined in gsi_sminres/extras/sparse/csr.hpp.
[in,out]xSolution vector; on input an initial guess, on output the approximate solution.
[in]bRight-hand side vector.
[in]max_iterMaximum number of iterations.
[in]rtolRelative residual tolerance.
[out]itersNumber of iterations performed (set even if the method does not converge).
[out]relresFinal relative residual \( \|r\|_2 / \|b\|_2 \) at exit.
Returns
true if the tolerance is satisfied within max_iter; false otherwise.
Precondition
A and B are stored in CSR format as defined in gsi_sminres/extras/sparse/csr.hpp.
A and B are square and real-symmetric or Hermitian.
\( \omega \in \mathbb{R} \).
