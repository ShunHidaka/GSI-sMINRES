Detailed Description

Demo: Shift–Invert preconditioned shifted MINRES with BLAS/LAPACK packed-Hermitian (ZHP) backends.

Solves \((A + \sigma^{(m)} B)x^{(m)} = b\) using the Shift–Invert preconditioned shifted MINRES solver. Both A and B are read into BLAS/LAPACK packed-Hermitian (ZHP) storage; \(S = A + \omega B\) is factorized by zhptrf and applied via zhptrs every outer iteration.

Input

  • A.mtx (Hermitian / symmetric, ZHP after loading)
  • B.mtx (Hermitian positive definite, ZHP after loading)
  • The program enforces uploA == uploB to keep storage consistent.

Output (per shift m)

  • index, Re(σ), Im(σ), iteration count, algorithmic residual estimate, and the true 2-norm residual \(\|(A+\sigma B)x-b\|_2\).
Usage
a_sis_zhp.out A.mtx B.mtx
Note
Requires BLAS/LAPACK; uses zhpmv for Hermitian MVs and zhptrf/zhptrs for solving with \(A + \omega B\).
