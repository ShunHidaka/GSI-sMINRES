Detailed Description

I/O helpers: Matrix Market (symmetric/Hermitian) -> Compressed Sparse Row (CSR) for GSI-sMINRES samples.

Author
Shuntaro Hidaka

Loads a real-symmetric or complex-Hermitian square matrix given in Matrix Market coordinate format and returns a struct 'CSRMatrix'. Indices are converted to 0-based, and duplicate entries are accumulated. On any error this function prints a message and terminates the process.
