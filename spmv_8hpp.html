Detailed Description

Sparse matrix–vector multiplication (SpMV): y <- A x for CSRMatrix.

Minimal SpMV used by GSI-sMINRES samples.

  • Input: A : CSRMatrix (square, 0-based indices) x : input vector (length == A.n)
  • Output: y : result vector (length == A.n), overwritten with A*x
  • Parallelism: Row-wise OpenMP parallelization; each iteration writes to distinct y[i].
  • Preconditions (not checked here for overhead minimization):
    • A.row_ptr.size() == A.n + 1
    • A.col_idx.size() == A.values.size() (== nnz)
    • x.size() == A.n, y.size() == A.n
    • 0 <= A.col_idx[k] < A.n for all nonzeros
  • Notes: This kernel assumes a general CSR (no symmetry expansion).
