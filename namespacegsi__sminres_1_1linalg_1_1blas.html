Detailed Description

[Utility] This namespace provides C++ wrappers for BLAS routines used in GSI-sMINRES sample programs.

This namespace provides simple C++ wrappers over the BLAS routines. These routines are used in GSI-MINRES.

Hermitian packed 'U' matrix-vector multiplication: \( y = \alpha A x + \beta y \).

Parameters
[in]uploWhich triangle of A is stored: 'U' (upper) or 'L' (lower).
[in]nProblem dimension (length of logical vectors x and y).
[in]alphaScalar multiplier for A*x.
[in]apPacked Hermitian storage of A, size \( n(n+1)/2 \). The data are assumed contiguous and start at ap.data().
[in]xInput vector buffer (containing x).
[in]x_offsetElement offset to the first logical entry of x (0-based).
[in]betaScalar multiplier for y.
[in,out]yIn/out vector buffer (overwritten with the result).
[in]y_offsetElement offset to the first logical entry of y (0-based).
[in]incxStride (in elements) between consecutive entries of x (default 1).
[in]incyStride (in elements) between consecutive entries of y (default 1).
