Detailed Description

Demo: Generalized shifted MINRES (CSR backends; inner solves by CG).

Solves \((A + \sigma^{(m)} B)x^{(m)} = b\) for multiple complex shifts. Both A and B are loaded in CSR. The generalized Lanczos steps apply \(B^{-1}\) via a CG inner solve (tolerance can be tuned) and use CSR SpMV for A and B.

Input

  • A.mtx (Hermitian / real-symmetric, CSR after loading)
  • B.mtx (Hermitian / real-symmetric positive definite, CSR after loading)

Output (per shift m)

  • index, Re(Ïƒ), Im(Ïƒ), iteration count, algorithmic residual estimate, and the true 2-norm residual \(\|(A+\sigma B)x-b\|_2\).
Usage
a_gen_csr.out A.mtx B.mtx
Note
Inner CG iterations and relative residual are reported inside the code (variables available if you wish to print them).
