Detailed Description

Top-Level namespace of the GSI-MINRES library.

Major sub-namespaces:

  • generalized : generalized shifted minres solver interface
  • shift_invert : shift-invert precondition shifted minres solver interface

  • linalg::{blas,lapack} : linear algebra wrappers
  • io::{mm,csr} : I/O utilities for easy-to-use GSI-MINRES
