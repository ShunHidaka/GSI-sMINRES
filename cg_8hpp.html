Detailed Description

Unpreconditioned Conjugate Gradient (CG) solver; compatible with CSR storage.

Author
Shuntaro Hidaka

Declares a CG routine that operates on matrices stored in the CSR format as defined in gsi_sminres/extras/sparse/csr.hpp. Assumes the coefficient matrix is real-symmetric or Hermitian positive definite. The input vector x is used as an initial guess and overwritten with the approximate solution.
