Detailed Description

I/O helpers: Matrix Market (symmetric/Hermitian) -> BLAS packed (ZHP) for GSI-sMINRES samples.

Author
Shuntaro Hidaka

Declares a loader that converts a Matrix Market file describing a symmetric/Hermitian square matrix into the BLAS/LAPACK complex Hermitian packed storage (ZHP). The packed layout is chosen as either upper ('U') or lower ('L').
