Detailed Description

Demo: Shift–Invert preconditioned shifted MINRES with CSR backends.

Solves a batch of shifted linear systems \((A + \sigma^{(m)} B) x^{(m)} = b \quad (m=0,\dots,M-1)\) using the Shift–Invert preconditioned shifted MINRES solver. Matrices A and B are loaded from Matrix Market files and stored in CSR.

The shift–invert preconditioner applies \((A + \omega B)^{-1}\) to vectors; this example sets \(\omega=0\) and uses a helper MINRES routine on the matrix pencil \((A, B)\) in CSR to realize the solve.

Input

  • A.mtx (Hermitian / real-symmetric, CSR after loading)
  • B.mtx (Hermitian / real-symmetric positive definite, CSR after loading)

Output (per shift m)

  • index, Re(σ), Im(σ), iteration count, algorithmic residual estimate, and the true 2-norm residual \(\|(A+\sigma B)x-b\|_2\).
Usage
a_sis_csr.out A.mtx B.mtx
Note
This is a minimal end-to-end example intended for correctness checks, not a performance benchmark. The solution array x is laid out as M contiguous length-N blocks (one block per shift).
