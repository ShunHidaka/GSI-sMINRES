[Solver] Standard shifted MINRES method.

The standard namespace provides the core infrastructure for solving multiple standard shifted linear systems using the standard shifted MINRES method.

It includes:

  • The Solver class for orchestrating the standard shifted MINRES iteration.
  • Numerical routines interfacing with BLAS and LAPACK.
  • Utility functions and data structures for managing matrices and vectors.
