Detailed Description

Demo: Standard shifted MINRES (no B) with BLAS packed-Hermitian MVs.

Solves \((A + \sigma^{(m)} I)x^{(m)} = b\) using the standard shifted MINRES algorithm. Matrix A is read into BLAS packed-Hermitian (ZHP) storage and applied by zhpmv each iteration.

Input

  • A.mtx (Hermitian / symmetric, ZHP after loading)

Output (per shift m)

  • index, Re(σ), Im(σ), iteration count, algorithmic residual estimate, and the true residual norm \(\|(A+\sigma I)x-b\|_2\).
Usage
a_std_zhp.out A.mtx
